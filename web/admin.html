Per-user activity counts. Favs = total favorites. Sugg pool = current personal-suggestion row count. Sugg fav'd = how many of those suggestions the user has actually favorited. Album clicks = release/master modal opens (last 30d / lifetime). Plays = media-player track plays (last 30d / lifetime). Searches = Discogs main-search calls (last 30d / lifetime).

Unavailable Videos

Videos auto-reported as broken when the YouTube IFrame Player throws error code 100 (not found), 101, or 150 (embed disabled / region-blocked). After 2 distinct reports, the videoId graduates to "unavailable" and is filtered out of every album popup site-wide so the affected tracks register as missing and contributors can submit replacements.
